# LearnHub

LearnHub is a full-stack E-Learning platform that connects instructors and students in one place. The platform allows instructors to create and manage courses, while students can enroll, learn, and track their progress.

## Features

- User Authentication (Login & Register)
- Student & Instructor Roles
- Course Management
- Video Lessons
- Progress Tracking
- Course Enrollment
- Responsive Design
- Real-Time Notifications
- Dashboard for Users

## Tech Stack

### Frontend
- React
- Tailwind CSS
- React Router
- Context API / Redux Toolkit
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Socket.io
